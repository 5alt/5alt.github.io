git 搜索历史文件中出现的字符串

git log -S/-G/--all

  • -S 字符串方式查找改变了行数的不同的地方
  • -G 正则的方式匹配所有的改变

git log -Sbar -- foo.rb

在 foo.rb 文件中搜所有的历史
